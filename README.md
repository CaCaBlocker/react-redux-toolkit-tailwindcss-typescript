## Commands

🛠 How to run in local

```
npm run dev
```

🛠 How to link the code

```
npm run lint
```

🛠 How to check the format of code

```
npm run format:check
```

🛠 How to format the code

```
npm run format
```

🛠 How to commit the code

```
npm run commit
```

🛠 How to build

```
npm run build
```
